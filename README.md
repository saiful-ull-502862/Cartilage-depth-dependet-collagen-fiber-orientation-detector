# Integrated Cartilage Analysis Platform

A comprehensive platform for depth-dependent collagen fiber orientation analysis in articular cartilage. This repository contains tools for image analysis (homogenization, zone detection) and data visualization.

## Modules

The platform consists of two main applications:

1.  **Cartilage Analysis App**: 
    *   **Purpose**: Processes microscopy images (e.g., PLM) to detect cartilage zones (Superficial, Middle, Deep) using AI and calculates collagen fiber orientations.
    *   **Features**:
        *   AI-based Zone Segmentation.
        *   Fiber Orientation Analysis using Structure Tensor or similar methods.
        *   Generation of depth profiles.
    *   **Location**: `./cartilage_analysis_app`

2.  **Excel Plotter App**:
    *   **Purpose**: A dedicated tool for visualizing and comparing the output data (Excel files) generated by the analysis app.
    *   **Features**:
        *   Interactive plotting of depth profiles.
        *   Batch analysis comparison.
        *   Customizable charts.
    *   **Location**: `./excel_plotter_app`

## Prerequisites

*   Python 3.8 or higher
*   Git

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/saiful-ull-502862/Cartilage-depth-dependet-collagen-fiber-orientation-detector.git
    cd Cartilage-depth-dependet-collagen-fiber-orientation-detector
    ```

2.  **Create a Virtual Environment (Recommended):**
    ```bash
    python -m venv .venv
    # Windows
    .venv\Scripts\activate
    # Linux/Mac
    source .venv/bin/activate
    ```

3.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

### Running the Cartilage Analysis App
1.  Navigate to the app directory:
    ```bash
    cd cartilage_analysis_app
    ```
2.  Run the Flask application:
    ```bash
    python app.py
    ```
3.  Open your browser and go to `http://localhost:5000` (or the port displayed in the terminal).
4.  Upload an image from the `Sample_Data` folder to test.

### Running the Excel Plotter App
1.  Navigate to the app directory:
    ```bash
    cd excel_plotter_app
    ```
2.  Run the Flask application:
    ```bash
    python app.py
    ```
3.  Open your browser and navigate to the displayed port (usually `5001`).
4.  Upload the `fibril_angle_depth_profile1.xlsx` from `Sample_Data` to verify functionality.

## Sample Data

A `Sample_Data` directory is provided in the root to help you get started. It contains:
